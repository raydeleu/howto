= Quick Reference for Markdown

== Headers

 #    header 1

 ###  header 3
 
== Emphasis 


 *This text will be italic*
 _This will also be italic_

 **This text will be bold**
 __This will also be bold__

 _You **can** combine them_

== Lists

=== Unordered
```
  * apples
  ** red apples
  ** green apples 
  * oranges
  * pears
```

* apples
** red apples
** green apples 
* oranges
* pears
 
=== Ordered
```
 . apples
 .. red apples
 .. green apples
 . oranges
 . pears
```

. apples
.. red apples
.. green apples
. oranges
. pears
